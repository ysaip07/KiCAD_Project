A Smart Car Parking System PCB designed using KiCad. This project integrates sensor interfacing, microcontroller logic, and actuator control to automate parking slot detection and management — demonstrating a complete embedded hardware design from schematic to fabrication-ready output.

System Description
- The Smart Car Parking System detects vehicle presence in each parking slot using IR or ultrasonic sensors and displays real-time slot availability. A central microcontroller processes sensor data and drives visual indicators (LEDs / display) to guide drivers to available slots.

System Flow:
- Sensors → Microcontroller → Decision Logic → Display / Indicators → User

Features
- Multi-slot vehicle presence detection using IR / Ultrasonic sensors
- Microcontroller-based slot availability logic
- LED / 7-segment display output for slot status indication
- Servo motor control for entry/exit gate automation
- Modular PCB design — sensor board + control board
- 5V power supply with onboard regulation
