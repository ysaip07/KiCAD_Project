A PCB design for an Astable Multivibrator circuit built using KiCad. This project demonstrates core analog electronics principles — the circuit generates a continuous square wave output without any external triggering, making it ideal for clock generation, LED blinking, and tone generation applications.
Circuit Description
An Astable Multivibrator is a free-running oscillator built using two transistors (or a 555 timer) in a cross-coupled feedback configuration. The output continuously switches between HIGH and LOW states at a frequency determined by the RC time constants.

Key Parameters:
- Oscillation frequency determined by R and C values
- Symmetric or asymmetric duty cycle support
- No external trigger required — self-starting circuit

Features
- Clean PCB layout with proper component placement
- Schematic capture with labeled nets and values
- Gerber files ready for PCB fabrication (JLCPCB / PCBWay compatible)
- Design Rule Check (DRC) passed with zero errors
